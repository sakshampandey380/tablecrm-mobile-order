# 🚀 CRM Sales Dashboard

A modern and responsive **CRM sales dashboard web app** built with **Next.js 15, TypeScript, Tailwind CSS, and Zustand**.

This project provides a clean UI, dynamic data tables, and customizable components for managing sales workflows efficiently.

---

## ✨ Features

* 📊 Interactive data tables using TanStack Table
* 🎨 Beautiful UI with Tailwind CSS & Radix UI
* 🌗 Light/Dark mode support
* ⚡ Fast performance with Next.js
* 🧩 Modular and reusable components
* 🔄 State management using Zustand
* 📱 Fully responsive design
* 🛠 Customizable dashboard layout

---

## 🛠 Tech Stack

* **Frontend:** Next.js 15 (React 18)
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **State Management:** Zustand
* **UI Components:** Radix UI, Lucide Icons
* **Table:** TanStack React Table
* **HTTP Client:** Axios

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run development server

```bash
npm run devsales dashboard web app
```

### 4. Open in browser

```
http://localhost:3000
```

---

## 🔐 Environment Variables

Create a `.env.local` file and add:

```
NEXT_PUBLIC_API_URL=your_api_url_here
```

---

## 🚀 Build for Production

```bash
npm run build
npm start
```
